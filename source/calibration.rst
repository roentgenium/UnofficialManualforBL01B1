エネルギー較正
==============

QXAFS を利用する場合
--------------------

操作
~~~~

1. エネルギー較正に適当な試料を QXAFS で測定する．
2. エネルギー較正の基準となるエネルギーおよび結晶の角度を求める．
3. Move |theta| ツールにおいて，**Encorder Board** を選択した状態でエネルギー較正の基準となる結晶の角度まで移動する．
4. エンコーダ表示器で「正しい」エネルギーに相当する角度を入力する．
5. Move\ |theta| ツールにおいて，**Encorder Board** を選択した状態で **Encorder Board Reset** を **ON** にして，実行する．これで **Encorder Board** が「正しい」角度になる．

XAFS Measure を利用する場合
---------------------------

操作
~~~~

1. エネルギー較正に適当な試料を XAFS Measure で測定する．
2. エネルギー較正の基準となるエネルギーおよび結晶の角度を求める．
3. Move |theta| ツールにおいて，**Encorder Display** を選択した状態でエネルギー較正の基準となる結晶の角度まで移動する．
4. エンコーダ表示器で「正しい」エネルギーに相当する角度を入力する．
5. **QXAFS も利用する場合**\ は，Move\ |theta| ツールにおいて，**Encorder Display** を選択した状態で **Encorder Board Reset** を **ON** にして，実行する．これで **Encorder Board も**\ 「正しい」角度になる．
